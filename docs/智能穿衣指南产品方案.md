## 一、产品概述

### 1.1 产品定位

**ClosetAI** 是一款基于AI技术的智能穿衣指南应用，以移动端H5为初始载体，帮助用户实现：

- **衣橱数字化**：AI自动识别、分类、卡通化衣物
- **智能穿搭推荐**：结合实时天气与个人衣橱，生成每日穿搭方案
- **虚拟试衣体验**：个性化虚拟模特，可视化呈现搭配效果

### 1.2 核心价值主张

| 痛点 | 解决方案 |
|------|---------|
| 衣橱满满却觉得没衣服穿 | AI智能推荐，盘活现有衣物 |
| 早晨时间紧，穿搭决策疲劳 | 3秒获取今日穿搭方案 |
| 不确定天气该穿什么 | 实时天气数据+智能算法 |
| 买了衣服不会搭配 | 虚拟模特直观展示效果 |

### 1.3 产品形态演进

| Phase 1 (MVP) | Phase 2 | Phase 3 |
|---------------|---------|---------|
| 移动端H5      | 微信小程序 | Native App |
| 核心闭环验证   | 社交分享裂变 | 完整生态 |

#### 4.1.2 核心交互流程

```mermaid
用户打开H5 → 自动定位获取天气 → 加载用户衣橱数据
    → AI计算推荐算法 → 生成3套方案 → 默认展示方案A
    → 用户切换/确认 → 记录用户选择 → 优化后续推荐
```

#### 4.1.3 推荐算法逻辑（MVP版）

```python
# 伪代码：基于规则的推荐引擎
def generate_outfit(weather, wardrobe, occasion):
    # 1. 天气过滤
    suitable_items = filter_by_temperature(wardrobe, weather.temp)
    suitable_items = filter_by_rain(suitable_items, weather.rain_prob)

    # 2. 场合过滤
    occasion_items = filter_by_occasion(suitable_items, occasion)

    # 3. 搭配组合生成
    combinations = generate_combinations(occasion_items)

    # 4. 评分排序
    scored_outfits = []
    for outfit in combinations:
        score = calculate_score(outfit, {
            'weather_match': 0.4,    # 天气适配度
            'color_harmony': 0.3,    # 色彩和谐度
            'occasion_fit': 0.2,     # 场合适配度
            'variety_bonus': 0.1     # 避免重复奖励
        })
        scored_outfits.append((outfit, score))

    return sorted(scored_outfits, key=lambda x: x[1], reverse=True)[:3]
```

### 5.3 关键技术方案

#### 5.3.1 衣物识别方案

**第一阶段：第三方API + 规则引擎**

- 调用阿里云/腾讯云图像识别API获取基础标签
- 自建映射表将通用标签映射到时尚分类体系
- 准确率预期：70-80%

**第二阶段：自研模型微调**

- 基于Fashion-MNIST、DeepFashion等数据集
- 使用CLIP或自研CNN模型进行细分类
- 目标准确率：85%+

#### 5.3.2 漫画生成方案

**技术选型**：Stable Diffusion + LoRA

- **基础模型**：动漫风格大模型（如Counterfeit、Pastel-Mix）
- **自研LoRA**：训练统一风格的衣物卡通化模型
- **训练数据**：500-1000张高质量成对数据（实物图+卡通图）
- **风格定位**：日系清新、线条简洁、色彩明快
- **触发词**：closetai style, cartoon clothes, flat lay
- **ControlNet**：使用Canny/Lineart控制轮廓，保持衣物结构

**生成参数模板**

```json
{
  "prompt": "closetai style, cartoon clothes, {{category}}, {{color}}, {{material}}, flat lay presentation, clean lines, soft pastel colors, minimalist background, kawaii aesthetic, high quality, detailed texture, white background, simple background",
  "negative_prompt": "realistic, photorealistic, messy background, dark colors, distorted, blurry, low quality, worst quality, bad anatomy, human, face, person",
  "steps": 25,
  "cfg_scale": 7,
  "sampler": "DPM++ 2M Karras",
  "size": "512x512"
}
```
