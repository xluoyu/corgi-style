## 二、唯一需要改进的问题

### 问题：多轮追问一次问多个

当前 `_build_missing_question` 将所有缺失信息一次性追问，体验不佳：

```
用户: "帮我推荐穿搭"
AI: "好的！想帮您搭配得更好，要去哪里（城市）呢？是什么场合（比如上班、约会、运动）？让我看看您的衣柜里有什么衣服～？"  ← 三个问题一次抛出
```

### 方案：逐个追问，一次只问一个

按优先级依次检查，缺失哪个就只问哪个。

## 三、实施（5 行代码）

**文件：`service/app/agent/graph/nodes/response.py`**

将 `_build_missing_question` 函数改为：

```python
def _build_missing_question(state: GraphState, intent_str: str) -> Optional[tuple[str, Dict]]:
    """
    检测缺失的关键信息，逐个追问（一次只问一个）。

    Returns:
        (question_text, data_dict) 如果需要追问，否则 None
    """
    if intent_str != "generate_outfit":
        return None

    # 按优先级逐一检查，一次只问一个问题
    if not state.get("target_city"):
        question = "好的！您要去哪个城市呢？"
        return question, {"type": "ask_info", "missing": ["city"], "asking_for": "city"}

    if not state.get("target_scene"):
        city = state.get("target_city", "")
        question = f"好的，{city}不错～您是去什么场合呢？（比如上班、约会、运动）"
        return question, {"type": "ask_info", "missing": ["scene"], "asking_for": "scene"}

    if not state.get("user_clothes"):
        question = "让我看看您的衣柜里有什么衣服～"
        return question, {"type": "ask_info", "missing": ["wardrobe"], "asking_for": "wardrobe"}

    return None
```

**预期效果**：

```
用户: "帮我推荐穿搭"
AI: "好的！您要去哪个城市呢？"

用户: "北京"
AI: "好的，北京不错～您是去什么场合呢？（比如上班、约会、运动）"

用户: "约会"
AI: [生成穿搭方案]
```

## 四、测试

```python
# service/test/test_response.py

class TestBuildMissingQuestion:
    """逐个追问测试"""

    def test_single_question(self):
        """应该一次只问一个问题"""
        state = {"target_city": None, "target_scene": None, "user_clothes": None}
        question, data = _build_missing_question(state, "generate_outfit")

        # 应该只问城市
        assert "哪个城市" in question
        assert data["asking_for"] == "city"
        assert question.count("？") == 1  # 只有一个问号

    def test_priority_order(self):
        """追问优先级：城市 > 场景 > 衣柜"""
        state = {"target_city": "北京", "target_scene": None, "user_clothes": None}
        question, data = _build_missing_question(state, "generate_outfit")
        assert "场合" in question
        assert data["asking_for"] == "scene"

        state = {"target_city": "北京", "target_scene": "work", "user_clothes": None}
        question, data = _build_missing_question(state, "generate_outfit")
        assert "衣柜" in question
        assert data["asking_for"] == "wardrobe"
```
