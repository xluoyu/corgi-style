# Corgi Style 项目开发进度文档

> 本文档供后续接手开发者使用，记录了完整的需求、设计、进度和待办事项。

## 项目概述

**项目名称**: Corgi Style - AI 穿搭智能助手
**技术栈**: Next.js 15 + React 18 (前端) | FastAPI + PostgreSQL + LangGraph (后端)
**项目路径**: `/d/Work/my/Corgi-style`

### 核心功能
1. 用户管理（基于设备指纹识别）
2. 衣橱管理（图片上传、AI分析、卡通化）
3. AI穿搭推荐（基于天气、场景、用户偏好）
4. 穿搭历史记录

```

DB_PORT=5432
DB_USER=postgres
```

### 代码风格
- 前端: TailwindCSS + Motion动画
- 后端: FastAPI + SQLAlchemy
- 遵循 AGENTS.md 中的设计规范（反主流美学）

### 常见问题排查
- **API连接失败**: 检查后端服务是否启动，检查 .env 配置
- **图片上传失败**: 检查阿里云OSS配置
- **AI功能不可用**: 检查 OPENAI_API_KEY 配置
