# 智能穿衣指南 H5 原型设计提示词

## 一、项目概述

### 1.1 产品信息
- **产品名称**：ClosetAI / 衣橱管家
- **Slogan**：你的AI造型师，让每一天穿搭零烦恼
- **产品定位**：基于AI技术的智能穿衣指南应用，移动端H5为载体
- **核心功能**：衣橱数字化、智能穿搭推荐、虚拟试衣体验

### 1.2 设计要求
- **输出格式**：单个完整的HTML文件（包含内联CSS和JavaScript）
- **交互要求**：可交互式原型，模拟真实应用体验
- **适配要求**：移动端优先，响应式设计（375px-414px宽度为主）
- **风格定位**：日系清新、简约时尚、温暖治愈

### 2.2 色彩系统

| 色彩角色 | 色值 | 用途 |
|---------|------|------|

| **辅助色** | #4ECDC4 (薄荷绿) | 次要按钮、成功状态、图标 |

| **卡片背景** | #FFFFFF (纯白) | 内容卡片 |
| **文字主色** | #2D3436 (深灰) | 标题、正文 |
| **文字次色** | #636E72 (中灰) | 辅助文字、说明 |
| **边框色** | #DFE6E9 (浅灰) | 分割线、边框 |

#### 4.1.2 设计要点

**天气看板**
- 背景：浅蓝色渐变，营造天空感
- 动画：天气图标有轻微动画（如太阳旋转、云朵飘动）
- 信息层次：温度 > 天气状况 > 其他信息

**推荐卡片**
- 背景：白色卡片，圆角16px
- 阴影：柔和的投影 `box-shadow: 0 4px 20px rgba(0,0,0,0.08)`
- 虚拟模特：卡通风格，占位图可用插画代替
- 评分标签：胶囊形状，不同颜色区分

**交互效果**
- 「换一套」按钮：点击后卡片有翻转动效
- 「就这套」按钮：点击后弹出确认动画
- 方案卡片：点击切换到对应方案
