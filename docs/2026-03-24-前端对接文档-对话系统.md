# AI 穿搭对话系统 - 前端对接文档

## 概述

AI 穿搭对话系统基于 LangGraph 多 Agent 编排，支持自然语言交互、天气查询、衣柜检索、智能穿搭推荐等功能。

**Base URL**: `http://localhost:8000`

---

## 一、核心 API

### 1.1 发送对话消息

**Endpoint**: `POST /chat/message`

**请求体**:
```typescript
interface ChatMessageRequest {
  user_id: string;      // 用户 UUID
  session_id?: string;   // 会话 ID，不传则创建新会话
  message: string;       // 用户消息
  context?: object;      // 可选附加上下文
}
```

**响应体**:
```typescript
interface ChatMessageResponse {
  session_id: string;    // 会话 ID
  message: string;       // 兼容旧版本：文本消息
  contents: ContentItem[]; // 新版本：多种内容格式
  data?: object;         // 额外数据
  suggestions?: Suggestion[];
}
```

---

## 二、响应内容格式（contents）

`contents` 是一个数组，每个元素是一种内容类型，支持多种格式组合返回。

### 2.1 文本内容 (type: "text")

```json
{
  "type": "text",
  "content": "为您推荐以下穿搭方案..."
}
```

### 2.2 穿搭卡片 (type: "outfit_card")

```json
{
  "type": "outfit_card",
  "content": {
    "plan": {
      "plan_id": "plan_001",
      "description": "春秋时节 职场商务",
      "items": {
        "top": {"color": "白色", "style": "简约"},
        "pants": {"color": "深蓝色", "style": "修身"}
      }
    },
    "clothes": {
      "top": {"id": "xxx", "category": "top", "color": "白色", "image_url": "..."},
      "pants": {"id": "xxx", "category": "pants", "color": "深蓝色", "image_url": "..."}
    },
    "match_score": 85.0,
    "scene": "work",
    "temperature": 18.5,
    "city": "北京"
  }
}
```

### 2.3 推荐问题 (type: "suggestions")

```json
{
  "type": "suggestions",
  "content": [
    {"type": "action", "text": "换个更休闲的版本", "action": "换个休闲点的风格"},
    {"type": "action", "text": "这套不太合适", "action": "不太满意，换一套"}
  ]
}
```

### 2.4 图片内容 (type: "image")

```json
{
  "type": "image",
  "content": {
    "url": "https://example.com/image.jpg",
    "alt": "穿搭示意图"
  }
}
```

### 2.5 商品卡片 (type: "product_card") - 预留

```json
{
  "type": "product_card",
  "content": {
    "product_id": "xxx",
    "title": "相似款推荐",
    "image_url": "https://example.com/product.jpg",
    "price": 299,
    "link": "https://shop.example.com/item/xxx"
  }
}
```

## 三、会话管理

### 3.1 获取会话状态

**Endpoint**: `GET /chat/session/{session_id}`

**响应**:
```json
{
  "session_id": "xxx",
  "user_id": "xxx",
  "created_at": "2026-03-24T10:00:00",
  "last_active": "2026-03-24T10:30:00",
  "context": {
    "target_city": "北京",
    "target_scene": "party",
    "target_temperature": 12.0
  }
}
```

### 3.2 删除会话

**Endpoint**: `DELETE /chat/session/{session_id}`

### 3.3 清除上下文

**Endpoint**: `POST /chat/session/{session_id}/clear`

清除上下文但保留对话历史。

## 四、典型对话流程

### 4.1 生成穿搭推荐

**请求 1**:
```json
POST /chat/message
{
  "user_id": "用户UUID",
  "message": "我后天想去北京参加晚宴"
}
```

**响应 1**:
```json
{
  "session_id": "新建的session_id",
  "contents": [
    {
      "type": "text",
      "content": "好的，后天（3月26日）北京参加晚宴，我来为您查询天气并生成穿搭方案..."
    }
  ]
}
```

**请求 2** (同一 session 继续对话):
```json
POST /chat/message
{
  "user_id": "用户UUID",
  "session_id": "上一步返回的session_id",
  "message": "太正式了，换个休闲点的"
}
```

**响应 2**:
```json
{
  "session_id": "同一个session_id",
  "contents": [
    {
      "type": "text",
      "content": "好的，根据您的反馈，为您调整为空闲舒适的风格..."
    },
    {
      "type": "outfit_card",
      "content": {
        "plan": {...},
        "clothes": {...},
        "match_score": 80.0
      }
    },
    {
      "type": "suggestions",
      "content": [
        {"type": "action", "text": "再调整一下", "action": "不太满意"}
      ]
    }
  ]
}
```

### 4.2 查询衣柜

## 五、前端集成建议

### 5.1 组件设计

```typescript
// 1. ChatBubble 组件 - 显示单条消息
interface ChatBubbleProps {
  role: 'user' | 'assistant';
  contents: ContentItem[];
}

// 2. OutfitCard 组件 - 穿搭卡片展示
interface OutfitCardProps {
  plan: OutfitPlan;
  clothes: Record<string, ClothesItem>;
  matchScore: number;
  onFeedback?: (action: string) => void;
}

// 3. SuggestionChips 组件 - 推荐问题标签
interface SuggestionChipsProps {
  suggestions: Suggestion[];
  onClick: (action: string) => void;
}
```

### 5.2 内容渲染示例

```typescript
function renderContents(contents: ContentItem[]) {
  return contents.map((item, index) => {
    switch (item.type) {
      case 'text':
        return <TextMessage key={index} text={item.content} />;

      case 'outfit_card':
        return <OutfitCard key={index} {...item.content} />;

      case 'suggestions':
        return <SuggestionChips key={index} items={item.content} />;

      case 'image':
        return <ImageMessage key={index} {...item.content} />;

      case 'product_card':
        return <ProductCard key={index} {...item.content} />;

      default:
        return null;
    }
  });
}
```

### 5.3 Session 管理

```typescript
// 保存 session_id 到 localStorage
localStorage.setItem('chat_session_id', response.session_id);

// 下次打开时恢复
const sessionId = localStorage.getItem('chat_session_id');
```

### 5.4 错误处理

```typescript
try {
  const response = await fetch('/chat/message', {...});
  if (!response.ok) {
    throw new Error('请求失败');
  }
  // 处理响应...
} catch (error) {
  // 显示错误提示
  showToast('网络错误，请重试');
}
```

## 六、响应码说明

| 状态码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 404 | Session 不存在或已过期 |
| 500 | 服务器内部错误 |

---

## 七、Session TTL

- **有效期**: 3 天
- 3 天内相同 session 继续对话，上下文会被保留
- 超过 3 天不互动，session 自动过期，需要重新开始
